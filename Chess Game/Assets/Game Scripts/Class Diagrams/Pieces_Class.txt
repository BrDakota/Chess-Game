================ PROPERTIES ================
    position: This is the position of the piece
    team/color: This is the team of the piece
    rules: This will define the rules of play for each piece (hardcoded into each piece)

================ METHODS ================
    getPosition: This will get position of the piece
    setPosition: This will set the position of the piece
    setTeam: Sets the team of the piece
    getTeam: Gets the team of the piece

================ FUCNTIONS ================
    movePiece(): This function should move pieces
    capturePiece(): This function should allow the capturing of pieces
